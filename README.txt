This is the source code folder of "jbead", a bead
crochet ropes design software. See www.jbead.ch for
more information.

Originally called DB-BEAD, this software was written
in C++ using Borland C++Builder 5 and the VCL library,
later ported to Java 6 using the Swing GUI library.

This version is a Blazor WebAssembly port of the Java
app. Source lives under csharp/JBead.Web; open
jbead.sln in Visual Studio or run "dotnet run" from
csharp/JBead.Web to start the dev server.
